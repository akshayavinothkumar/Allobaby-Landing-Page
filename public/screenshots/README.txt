AlloBaby Screenshot Placeholders
================================

Active screenshot files are stored in:

  public/screenshots/

Replace PNG files there using these exact base names:

  - dashboard-placeholder
  - monitoring-placeholder
  - ai-assistant-placeholder
  - appointments-placeholder
  - consultation-placeholder
  - alerts-placeholder
  - baby-monitoring-placeholder
  - health-tracking-placeholder

See SCREENSHOT_REPLACEMENT_GUIDE.md in the project root for full instructions.
